    <h2>Projeto</h2>
    <p align="justify">Este é um projeto da 10º Imersão Dev Alura, uma linha do tempo interativa que mostra a história do hardware em 50 cards. Desenvolvido para praticar a manipulação de dados, o site utiliza um arquivo JSON padronizado, cuja estrutura e pesquisa foram criadas com a ajuda do Gemini. <br>
     O objetivo é mostrar a evolução do hardware de maneira bem resumida, apresentando desde as primeiras calculadoras gigantes até os componentes modernos. As informações podem ser filtradas por tipo ou marcas importantes, e ordenadas das mais antigas para as mais novas e vice-versa.<br>
     É uma viagem no tempo para entender as peças que formam o coração do nosso mundo digital. Espero que gostem.</p>
